## 最高级项目总纲

所有团队、Codex、GPT 的终极目标，是建立一条从客户原始平面图到最终报价成果的自动化生产线：

客户给原始平面图
→ 消防 CAD 自动画图系统自动生成消防各专业图纸
→ 系统从自动生成的图纸中统计设备、管线、点位、工程量
→ 输出标准工程量清单
→ 宏业自动报价系统导入工程量清单
→ 标准化解析
→ 匹配宏业定额数据库
→ 生成宏业标准版报价清单草稿
→ 生成人工复核表
→ 输出最终报价成果
→ 团队用这套生产线接单、交付、盈利。

项目分工：

- 消防 CAD 自动画图项目是前端生产系统，核心是自动绘制消防各专业图纸。
- 宏业自动报价项目是后端报价引擎，核心是根据工程量清单自动生成报价成果。
- 两个项目通过“标准工程量清单中间结构”打通。
- 不允许把 CAD 项目说成单纯读图算量。
- 不允许把宏业项目混入 CAD 绘图逻辑。
- 所有开发任务都必须服务于这条最终生产线。
