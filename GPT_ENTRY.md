# GPT_ENTRY - 最高级项目总纲

所有团队、Codex、GPT 的终极目标：

客户给原始平面图
→ 消防 CAD 自动画图系统自动生成消防各专业图纸
→ 从自动生成的图纸中统计设备、管线、点位、工程量
→ 输出标准工程量清单
→ 宏业自动报价系统导入工程量清单
→ 标准化解析
→ 匹配宏业定额数据库
→ 生成宏业标准版报价清单草稿
→ 生成人工复核表
→ 输出最终报价成果
→ 团队用这套生产线接单、交付、盈利。

## 项目分工

- 消防 CAD 项目是前端生产系统，核心是自动绘制消防专业图纸。
- 工程量清单是自动画图完成后的副产物，不是 CAD 项目的核心。
- 宏业项目是后端报价引擎，核心是工程量清单 → 定额匹配 → 报价清单 → 人工复核 → 成果输出。
- 两个项目通过标准工程量清单中间结构连接。

## 永久纠偏

- 禁止把 CAD 项目说成单纯读图算量。
- 禁止把宏业项目混入 CAD 绘图逻辑。
- 所有任务都必须服务于从平面图到报价成果的最终生产线。
