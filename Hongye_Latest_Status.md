# Hongye Latest Status

## 最高级目标

从客户原始平面图到最终报价成果的自动化生产线。

## 项目分工

- 消防 CAD 自动画图项目：前端生产系统，自动绘制消防专业图纸。
- 工程量清单：自动画图完成后的副产物，不是 CAD 项目的核心。
- 宏业自动报价项目：后端报价引擎，生成报价清单和人工复核表。
- 两个项目：通过标准工程量清单中间结构打通。

## 当前宏业项目定位

工程量清单 → 定额匹配 → 报价清单 → 人工复核 → 成果输出。

## 安全状态

公共状态仓库只同步状态、报告、下一步任务，不同步源码、原始 XLSX、output、宏业数据库、真实报价成果、token、key、密码。
